## 🚀 GitHub Pagesで公開する手順

### ステップ1: GitHubアカウントを作る
1. https://github.com にアクセス
2. 「Sign up」からアカウント作成（無料）

### ステップ2: リポジトリを作る
1. 右上の「+」→「New repository」をクリック
2. Repository name に好きな名前を入力（例：`love-message`）
3. 「Public」を選択
4. 「Create repository」をクリック

### ステップ3: ファイルをアップロード
1. 「uploading an existing file」リンクをクリック
2. ダウンロードしたファイルを全部ドラッグ＆ドロップ
3. 下の「Commit changes」をクリック

### ステップ4: GitHub Pagesを有効にする
1. リポジトリの「Settings」タブをクリック
2. 左メニューの「Pages」をクリック
3. Source を「Deploy from a branch」にする
4. Branch を「main」、フォルダを「/ (root)」にして「Save」

### ステップ5: URLを確認
1. 数分待つとURLが表示される
2. `https://あなたのID.github.io/リポジトリ名/` でアクセスできる！
3. このURLを恋人に送ってあげよう💕

## ✏️ メッセージの書き方

`messages.js` を開いて、""の中を書き換えるだけ！

```javascript
const messages = [
  "大好きだよ",
  "今日も一日がんばってね",
  "会いたいな",
  // ... 365個書いてね！
];
```

### メッセージ編集のコツ
- GitHubのサイト上で直接編集できる（ファイルをクリック→鉛筆アイコン）
- 各メッセージは `"` で囲む
- メッセージの間は `,` で区切る
- 最後のメッセージの後には `,` は不要

## 📸 写真を入れる方法

1. 二人の写真を用意
2. ファイル名を `photo.jpg` にリネーム
3. GitHubにアップロード（他のファイルと同じ場所に）

※ 写真がない場合は自動で💕マークが表示されます

## 📱 ホーム画面に追加する方法（恋人に教えてあげてね）

### iPhoneの場合
1. Safariでページを開く
2. 下の共有ボタン（↑マーク）をタップ
3. 「ホーム画面に追加」をタップ

### Androidの場合
1. Chromeでページを開く
2. 右上の「⋮」→「ホーム画面に追加」をタップ

---

## 🔄 メッセージを更新したい時

1. GitHubのリポジトリページを開く
2. `messages.js` をクリック
3. 右上の鉛筆アイコン（Edit）をクリック
4. メッセージを編集
5. 「Commit changes」をクリック
6. 数分後に反映される！

※ キャッシュが残ってる場合は、恋人のスマホでページを再読み込みしてもらってね
